Контекстный отчет предназначен для анализа данных партнера с точки зрения ключевых показателей:

  • Контактов;
  • Текущей активности;
  • Данных клиента (если партнер является клиентом);
  • Данных поставщика (если партнер является поставщиком);
  • Классификации.


В случае установки в отчете флага в поле Контакты будет выведена информация о всех имеющихся контактах партнера (включая вышестоящих контрагентов).

В случае установки флага в поле Текущая активность будет выведена информация о активных сделках, маркетинговых мероприятиях, проектах в которых партнер принимает участие.

В случае установки флага в поле Данные клиента будет выведена информация о незавершенных претензиях клиента, объемах его покупок, действующих коммерческих соглашениях, заказах в работе и данных по взаиморасчетам.

В случае установки флага в поле Данные поставщика будет выведена информация о соглашениях с поставщиком, заказах поставщику и состоянии взаиморасчетов с ним.

В случае установки флага в поле Классификация будет выведена информация о двух последних АВС/XYZ классификациях и сегментах в которые входит партнер.

Если данные о расчетах с партнером не актуальны, то становится доступной кнопка Актуализировать расчеты. При выполнении данной команды данные о расчетах будут актуализированы и отчет будет переформирован.
