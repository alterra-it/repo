Помощник быстрой настройки параметров учета

Помощник предназначен для быстрой настройки параметров учета, а также заполнения следующей информации (в пошаговом режиме):

  • Определить в какой валюте будет вестись управленческий и бухгалтерский учет и заполнить актуальные курсы.
  • Заполнить информацию об основной организации торгового предприятия, ввести все необходимые данные для начала ведения учета от имени данной организации.
  • Определить структуру предприятия: ввести информацию о складах, кассах и подразделениях торгового предприятия.
  • Определить правила продаж на торговом предприятии: использование соглашений, заказов клиентов, поставщиков и т.д.
  • Заполнить все необходимые классификаторы для начала ведения учета: справочник валют, классификатор единиц измерения, классификатор банков, календарь предприятия.

Если пользователь не добавляя данных переходит на следующую страницу работы помощника (кнопка Далее), то программа автоматически создает необходимые элементы с заполнением необходимых данных для учета.
Основная организация. Общая система налогообложения, метод оценки стоимости товаров Средняя за месяц.
Основной банковский счет. Валюта счета равна валюте регламентированного учета, номер счета "00..00".
Основная касса. Валюта кассы равна валюте регламентированного учета.
Основное подразделение.
Основной склад. Оптовый, остатки с учетом резерва, уровень обслуживания 84.0%.

Примечание. Информация о трех введенных элементах (например, три организации) показывается в помощнике отдельными ссылками. Если элементов больше трех, то тогда будет показана общая ссылка по открытию списка (например, списка организаций).
